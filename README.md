# Locus

Locus is a social media. <br/>
Locus will guide you to your cyber-place wher you will find like-minded individuals. <br/>
All topics are covered, there is no shortage of details. <br/>

The topics are organized like a tree. <br/>
With each branch is a sub-topic. <br/>
It's a lot of fun to hop from one topic to another. <br/>
But you can also dig deeper to find specific topics. <br/>

The topics are organized like a tree for the simplicity of use. <br/>
For example: <br/>
- I am passionate about [Anthropocentrism](https://en.wikipedia.org/wiki/Anthropocentrism).
- That philosophy puts humans and humans experience in the center.
- [Anthropocentrism](https://en.wikipedia.org/wiki/Anthropocentrism). could be discovered through [Environmental Ethics](https://en.wikipedia.org/wiki/Environmental_ethics).

And to realy visualize it here is the full branch: <br/>
root -> immaterial -> humancentric -> philosophy -> environmental ethics -> anthropocentrism <br/>
